## API Endpoints Map

```
HTTP Request Flow:

GET /
  вҶ’ Route Handler
  вҶ’ Render: home.ejs
  в””в”Җ Response: HTML (Home Page)

GET /products
  вҶ’ Route Handler
  вҶ’ Query: Product.find({})
  вҶ’ Render: products/index.ejs
  в””в”Җ Response: HTML (Product Grid)

GET /products/new
  вҶ’ Route Handler
  вҶ’ Render: products/new.ejs
  в””в”Җ Response: HTML (Form)

POST /products
  вҶ’ Route Handler
  вҶ’ Body: product[productName], product[price], etc.
  вҶ’ Save: new Product(req.body.product)
  вҶ’ Database: MongoDB
  вҶ’ Redirect: /products
  в””в”Җ Response: 302 Redirect

GET /products/:id
  вҶ’ Route Handler
  вҶ’ Query: Product.findById(id)
  вҶ’ Render: products/show.ejs
  в””в”Җ Response: HTML (Product Details)

GET /products/:id/interested
  вҶ’ Route Handler
  вҶ’ Query: Product.findById(id)
  вҶ’ Render: products/show.ejs with showSeller: true
  в””в”Җ Response: HTML (Seller Details)

PUT /products/:id
  вҶ’ Route Handler
  вҶ’ Update: Product.findByIdAndUpdate(id, {...})
  вҶ’ Database: MongoDB
  вҶ’ Redirect: /products/:id
  в””в”Җ Response: 302 Redirect

PATCH /products/:id/buy or /unsold
  вҶ’ Route Handler
  вҶ’ Update: product.sold = true/false
  вҶ’ Database: MongoDB
  вҶ’ Redirect: /products/:id
  в””в”Җ Response: 302 Redirect

DELETE /products/:id
  вҶ’ Route Handler
  вҶ’ Delete: Product.findByIdAndDelete(id)
  вҶ’ Database: MongoDB
  вҶ’ Redirect: /products
  в””в”Җ Response: 302 Redirect

GET /donate
  вҶ’ Route Handler
  вҶ’ Render: donate.ejs
  в””в”Җ Response: HTML (Donation Form)

POST /donate
  вҶ’ Route Handler
  вҶ’ Body: product[productName], product[seller], etc.
  вҶ’ Save: new Product(req.body.product)
  вҶ’ Database: MongoDB
  вҶ’ Render: donate-thank-you.ejs
  в””в”Җ Response: HTML (Thank You Page)
```
