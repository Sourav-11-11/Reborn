# Reborn 🛍️

Reborn is a full-stack product listing web application built using Node.js, Express, MongoDB, and EJS.  
It allows users to create, view, edit, and delete products with proper MVC architecture and RESTful routing.

---

## 🚀 Features

- Create new products
- View all products
- View single product details
- Edit product information
- Delete products
- Server-side rendering using EJS
- RESTful routes
- Clean MVC folder structure

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- CSS
- JavaScript

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/Sourav-11-11/Reborn.git

2. Navigate into the folder

cd Reborn

3. Install dependencies

npm install

4. Start the server

node app.js

5. Open browser

http://localhost:3000
